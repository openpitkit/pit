Module pretrade/policies

@openpit/engine/pretrade/policies - built-in policy builders.

Exposes the staged builders and barrier types for order validation, order-size limits, rate limits, PnL kill switches, and spot-funds checks, plus their build* factories. Order validation and limit-only spot funds are ready immediately; barrier-driven builders become tokens accepted by EngineBuilder.builtin after a barrier configuration call.
