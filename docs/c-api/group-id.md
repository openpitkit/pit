## `OPENPIT_DEFAULT_POLICY_GROUP_ID`

The default policy-group identifier used when a caller does not assign a policy
to a specific group. Mirrors `openpit::DEFAULT_POLICY_GROUP_ID`.

```c
#define OPENPIT_DEFAULT_POLICY_GROUP_ID ((uint16_t) 0)
```
