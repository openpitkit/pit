pre-trade risk python rust open source apache 2.0

Embeddable pre-trade risk pipeline. Designed to be embedded and adaptable. Deterministic. In-memory. Policy-driven.

Binary wheels for Linux, macOS, and Windows. Install with
pip install openpit
→ pypi.org/project/openpit/
crates.io
Rust SDK published on the official package registry. Add with
cargo add openpit
→ crates.io/crates/openpit
